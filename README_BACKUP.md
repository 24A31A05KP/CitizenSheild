# SecureShe - Women Safety Platform

SecureShe is a comprehensive women safety platform that provides emergency SOS alerts, incident reporting, safety resources, and helpline information.

## Features

- 🚨 **SOS Emergency Alert** - One-click SOS with live location sharing to emergency contacts
- 📍 **Live Location Sharing** - Share your real-time location with trusted contacts
- 📝 **Incident Reporting** - Anonymous reporting of unsafe zones
- 📚 **Safety Resources** - Self-defense guides, legal rights information, and safety tips
- 📞 **Helpline Directory** - Emergency numbers for different countries
- 👤 **User Dashboard** - Manage emergency contacts and view SOS history

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Python Flask
- **Database**: MySQL
- **APIs**: Twilio (SMS), Google Maps, Email
